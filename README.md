## Configuration

Add to your `claude_desktop_config.json` (Claude Desktop) or your MCP client config:

```json
{
  "mcpServers": {
    "meok-eu-aia-art-9-rms-mcp": {
      "command": "uvx",
      "args": ["meok-eu-aia-art-9-rms-mcp"]
    }
  }
}
```

Or: `pip install meok-eu-aia-art-9-rms-mcp` then run the `meok-eu-aia-art-9-rms-mcp` command (stdio transport).
